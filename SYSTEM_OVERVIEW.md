## ğŸ”„ How It Works

### For Team Members

1. **Monday**: 
   - Read the week's task from `TASKS.md`
   - Create folder: `tasks/week-XX/your-name/`
   - Copy `daily-goals.md` template
   - Plan and start coding

2. **Tuesday-Thursday**:
   - Work on daily goals
   - Update `daily-goals.md` each day
   - Push code to GitHub daily

3. **Friday**:
   - Complete the project
   - Finalize documentation
   - Submit for review

### For Reviewers (You)

1. **Daily (Mon-Thu)**:
   - Check GitHub for daily commits
   - Review `daily-goals.md` files
   - Update `WEEKLY_REVIEW.md` with progress
   - Identify any blockers early

2. **Friday**:
   - Review complete projects
   - Test applications
   - Provide feedback
   - Mark tasks as complete
